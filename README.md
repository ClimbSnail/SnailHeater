# SuperHeat
多功能加热控制器（二代），平板加热台、风枪双控。

* 本项目的地址 https://github.com/ClimbSnail/SuperHeat （最新的资料）
* 或者 https://gitee.com/ClimbSnailQ/SuperHeat （仅为预览，不一定是最新的）
* 旧版一代控制器项目 https://github.com/ClimbSnail/HeatPlatform_SMT （底成本方案）
